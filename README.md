## Requirements

- ROS Melodic
- Ubuntu 18.04

## Compilation

```bash
catkin_make
```

## Usage

First, start the ROS core:

```bash
roscore
```

Then, launch the node:

```bash
rosrun laser_scan_to_grid
```
